# Getting Started
## Hardware
Currently the system is running on:
```
Product: OptiPlex 9020
CPU: i7-4790
Memeory: 32GB
```

## Setup
```
git clone https://github.com/amarjandu/neptunes-orbit.git
cd neptunes-orbit
docker-compose up

# background mode
docker-compose up -d 
```
